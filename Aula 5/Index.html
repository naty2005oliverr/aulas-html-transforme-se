O HTML (HyperText Markup Language), ou Linguagem de Marcação de Hipertexto, é a base para a construção de páginas web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação usada para estruturar e organizar o conteúdo de um site. Através de tags e atributos, o HTML define como os elementos de uma página serão exibidos em navegadores.
